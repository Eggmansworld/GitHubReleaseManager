Copy your preferred batch file and place it beside the python script file. Now you can create a shortcut of the batch file and put it on your desktop or anywhere else for easy running of the script.

!Eggman's GitHub Release Manager.bat
 - console window runs behind the GUI window, and when the app is closed, the window pauses until you manually close it.

!Eggman's GitHub Release Manager (auto-close).bat
- same as above, only when you close the GUI window, the console window closes too.

!Eggman's GitHub Release Manager (no console).bat
- if you're certain you don't want a console window, use this.
