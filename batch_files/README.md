Select and copy a batch file and place it beside the python script file. Now you can create a shortcut of the batch file and use it for easy running of the script without needing to open a command window.

Eggman's GitHub Release Manager.bat
 - console window runs behind the GUI window and when the app is closed, the console window pauses and remains open until you manually close it.

Eggman's GitHub Release Manager (auto-close).bat
- same as above, except when you close the GUI window, the console window closes too.

Eggman's GitHub Release Manager (no console).bat
- if you're certain you don't need to see the console window, use this.
